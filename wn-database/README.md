# Déploiement sur Supabase

1/ Créer un nouveau projet (bien conserver le mot de passe de la base de données)

2/ Aller dans table editor et utiliser le script SQL dans init.sql pour créer les schemas et les différentes tables

3/ Dans le tableau de bord de la base de données, cliquer sur Copy puis 'Get Connected'. Dans l'onglet 'Direct', choisir 'Session pooler' et 'URI'. Récupérez en dessous les éléments dont vous avez besoin
